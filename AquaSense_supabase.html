Conexão direta com Supabase
Sem precisar de servidor Node.js! O AquaSense usa a API REST nativa do Supabase. Cole sua Project URL e a anon key (em Project Settings → API).
⚡ Supabase conectado
⚠️ Importante: Ative o acesso anônimo nas tabelas. No Supabase, vá em Authentication → Policies e crie uma policy "Enable all" para anon em cada tabela — ou desative o RLS temporariamente em Table Editor → [tabela] → RLS disabled.
Escolha um provedor para hospedar gratuitamente a API AquaSense com SQL Server ou banco compatível.
